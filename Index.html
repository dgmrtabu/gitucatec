Curso Basico de Git

Qué es Git

Git es una herramienta que realiza una función del control de versiones de código de forma distribuida

Qué es Github y qué es lo que le ofrece a los desarrolladores

Vamos a explicarte qué es exactamente Github y cuáles son sus características principales. Se trata de una de las principales plataformas para crear proyectos abiertos de herramientas y aplicaciones, y se caracteriza sobre todo por sus funciones colaborativas que ayudan a que todos puedan aportar su granito de arena para mejorar el código.

Esta basada en el sistema de control de versiones distribuida de Git, por lo que se puede contar con sus funciones y herramientas, aunque GitHub ofrece varias opciones adicionales y su interfaz es mucho más fácil de manejar, por lo que no es absolutamente necesario que las personas que lo usan tengan un gran conocimiento técnico.
